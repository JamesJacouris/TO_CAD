Export — ``src.export``
========================

FreeCAD macro for importing pipeline JSON into a FreeCAD document.

.. note::

   This module runs inside the FreeCAD Python interpreter.  It cannot be
   imported in a standard Python environment.  Sphinx mocks the FreeCAD
   API symbols; autodoc output may be incomplete for FreeCAD-specific code.

   See :ref:`guides/freecad_export:freecad export` for usage instructions.
