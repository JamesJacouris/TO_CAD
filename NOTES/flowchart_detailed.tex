\caption{\textbf{Detailed four-stage pipeline architecture.}
Stage~0 performs topology optimization via the SIMP method, producing
a voxel density field $\rho$ with boundary-condition tags $\mathbf{t}$.
Stage~1 extracts a skeletal graph $\mathbf{G}$ via medial-axis thinning
and consolidates BC tags to preserve load/support locations.
Stage~2 refines node positions $\mathbf{x}$ via frame finite-element
analysis and quasi-Newton optimization.
Stage~3 optimizes strut cross-section radii $\mathbf{r}$ subject to a
volume constraint, producing a CAD-ready parametric model.
Each stage outputs JSON with full design history.}
